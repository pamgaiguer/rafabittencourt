Facebook Public Photo Slider is a photo album sharing slider plugin for wordpress that allows users to share publicly available facebook photo album on a wordpress site.

== Description ==

You can share any public shared facebook photo album or fanpage album using this plugin. It doesn't require any registration or app Id or anything. All you need is the url link of the album that you want to share on your wordpress site.  But cannot share albums that have limited accessibity. For Live Demo, Please Visit [the demo page](http://www.binnash.com/facebook-public-photo-slider/) 

If you are looking for more customizable version, Please Check out Premium version at:[Binnash.com](http://www.binnash.com/facebook-public-photo-slider/)



== Installation ==

1. Upload `facebook-public-photo-lite.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings->Facebook Photo in wordpress admin panel
4. Grab the url link of publicly shared photo album from facebook.
5. Put the link in the admin page of the plugin and press "Generate Shortcode" button.
6. Copy the shortcode and paste in the page/post.

== Frequently Asked Questions ==

= Can I share my private photo albums or limited accessibility albums in facebook?  =

No. This plugin can show only those albums that can be browsed from facebook without logging in.
